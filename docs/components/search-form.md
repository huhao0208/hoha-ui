## 组件 API

### Attributes 属性

| 参数          | 说明          | 类型      |    可选值     | 默认值   |
|-------------|-------------|---------|:----------:|-------|
| loading     | 显示loading状态 | Boolean | true/false | false |
| icons       | 按钮图标配置      |         |            |       |
| searchParam | 默认搜索参数      | Object  |     /      | {}    |
| columns     | 配置数据        | Array|     /      | []    |


### Events 事件

| 事件名 | 说明      | 回调参数 |
|  ----  |---------|------|
| search | 点击按钮时触发 | {}   |
