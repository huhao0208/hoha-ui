# Icon 图标

## Props

| 属性名 | 说明 | 类型 | 默认值 |
| ------ | ---- | ---- | ------ |
| name | - | `string` | - |
| size | - | `IconSize` | 'medium' |
| color | - | `string` | '' |
| rotate | - | `number|string` | undefined |
| flip | - | `string` | undefined |
| inline | - | `boolean` | false |
| clickable | - | `boolean` | false |
