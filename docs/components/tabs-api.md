# Tabs 标签页

## Props

| 属性名 | 说明 | 类型 | 默认值 |
| ------ | ---- | ---- | ------ |
| modelValue | - | `string | number` | 0 |
| tabs | - | `TabItem[]` | () => [] |
| sticky | - | `boolean` | false |
| animated | - | `boolean` | true |
| swipeable | - | `boolean` | false |
| threshold | - | `number` | 0.3 |

## Events

| 事件名 | 说明 | 参数 |
| ------ | ---- | ---- |
| update:modelValue | - | `-` |
| change | - | `-` |
| disabled-click | - | `-` |

## Slots

| 插槽名 | 说明 |
| ------ | ---- |
| tab.id || index | - |
