# Button 按钮

## Props

| 属性名 | 说明 | 类型 | 默认值 |
| ------ | ---- | ---- | ------ |
| type | 按钮类型 | `ButtonType` | 'primary' |
| size | 按钮尺寸 | `ButtonSize` | 'medium' |
| disabled | 是否禁用 | `boolean` | false |
| loading | 是否加载中 | `boolean` | false |
