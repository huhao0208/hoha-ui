# 国际化

Hoha UI 提供完整的国际化支持，内置中/英/繁/日四种语言包，支持自定义扩展。

## 内置语言

| 语言 | 代码 | 说明 |
|------|------|------|
| 简体中文 | `zh-CN` / `zh` | 默认语言 |
| 繁体中文 | `zh-TW` / `zh-HK` | 繁体 |
| 英语 | `en-US` / `en` | 英语 |
| 日语 | `ja-JP` / `ja` | 日语 |

## 基础用法

### 全局配置

```typescript
import { createApp } from 'vue'
import HohaUI from '@hohaya/hoho'
import '@hohaya/hoho/style.css'

const app = createApp(App)

// 方式1: 通过安装选项设置
app.use(HohaUI, {
  locale: 'en-US'
})

app.mount('#app')
```

### 使用 i18n 选项

```typescript
app.use(HohaUI, {
  i18n: {
    locale: 'en-US',        // 当前语言
    fallbackLocale: 'zh-CN', // 回退语言
    messages: {             // 自定义消息
      'en-US': {
        hoha: {
          confirm: 'OK',
          cancel: 'Close'
        }
      }
    }
  }
})
```

## 在组件中使用

### Composition API

```vue
<script setup>
import { useLocale } from '@hohaya/hoho'

const { t, locale, setLocale } = useLocale()

// 获取翻译
const confirmText = t('hoha.confirm')

// 切换语言
const switchLanguage = (lang) => {
  setLocale(lang)
}
</script>

<template>
  <div>
    <p>{{ t('hoha.loading') }}</p>
    <button @click="switchLanguage('zh-CN')">中文</button>
    <button @click="switchLanguage('en-US')">English</button>
  </div>
</template>
```

### Options API

```vue
<script>
export default {
  methods: {
    getText() {
      // 通过全局属性访问
      return this.$t('hoha.confirm')
    }
  }
}
</script>
```

## API

### t(path, ...args)

翻译函数，支持参数替换。

```typescript
import { t } from '@hohaya/hoho'

// 基础用法
t('hoha.confirm')  // '确认'

// 带参数 (需要在语言包中定义 {0}, {1} 占位符)
t('hoha.selected', 3)  // '已选择 3 项'
```

### setLocale(locale)

切换当前语言。

```typescript
import { setLocale } from '@hohaya/hoho'

setLocale('en-US')
setLocale('zh-CN')
```

### addLocale(locale, messages)

添加或扩展语言包。

```typescript
import { addLocale } from '@hohaya/hoho'

// 扩展现有语言
addLocale('zh-CN', {
  hoha: {
    customText: '自定义文本'
  }
})

// 添加新语言
addLocale('ko-KR', {
  name: 'ko-KR',
  hoha: {
    confirm: '확인',
    cancel: '취소',
    loading: '로딩 중...'
  }
})
```

### getLocale()

获取当前语言配置。

```typescript
import { getLocale } from '@hohaya/hoho'

const locale = getLocale()
console.log(locale.name)  // 'zh-CN'
```

### getAvailableLocales()

获取所有可用语言列表。

```typescript
import { getAvailableLocales } from '@hohaya/hoho'

const locales = getAvailableLocales()
// ['zh-CN', 'zh', 'en-US', 'en', 'zh-TW', 'zh-HK', 'ja-JP', 'ja']
```

### createI18n(options)

创建独立的 i18n 实例。

```typescript
import { createI18n } from '@hohaya/hoho'

const i18n = createI18n({
  locale: 'en-US',
  fallbackLocale: 'zh-CN'
})

// 在 Vue 应用中使用
app.use(i18n)
```

## 自定义语言包

创建完整的自定义语言包：

```typescript
import { setLocale } from '@hohaya/hoho'

const thTH = {
  name: 'th-TH',
  hoha: {
    name: 'Hoha UI',
    confirm: 'ยืนยัน',
    cancel: 'ยกเลิก',
    loading: 'กำลังโหลด...',
    success: 'สำเร็จ',
    fail: 'ล้มเหลว',
    error: 'ข้อผิดพลาด',
    noData: 'ไม่มีข้อมูล',
    // ... 其他翻译
  }
}

setLocale(thTH)
```

## 动态切换语言

```vue
<template>
  <div class="language-switcher">
    <select v-model="currentLang" @change="changeLanguage">
      <option value="zh-CN">简体中文</option>
      <option value="en-US">English</option>
      <option value="zh-TW">繁體中文</option>
      <option value="ja-JP">日本語</option>
    </select>
  </div>
</template>

<script setup>
import { ref } from 'vue'
import { setLocale, useLocale } from '@hohaya/hoho'

const { locale } = useLocale()
const currentLang = ref(locale.value.name)

const changeLanguage = () => {
  setLocale(currentLang.value)
}
</script>
```

## 持久化语言设置

```typescript
import { setLocale, getLocale } from '@hohaya/hoho'

// 保存到 localStorage
const savedLocale = localStorage.getItem('hoha-locale')
if (savedLocale) {
  setLocale(savedLocale)
}

// 监听语言变化并保存
watch(() => getLocale().name, (newLocale) => {
  localStorage.setItem('hoha-locale', newLocale)
})
```
