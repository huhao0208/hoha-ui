# 国际化

Hoha UI 提供了国际化支持，允许你配置组件的语言。

## 基本用法

```typescript
import { createApp } from 'vue'
import HohaUI, { zhCn } from '@hoha/components'
import App from './App.vue'

const app = createApp(App)
app.use(HohaUI, {
  locale: zhCn
})
app.mount('#app')
```

## 可用语言

| 语言 | 语言包 | 说明 |
| --- | --- | --- |
| 简体中文 | `zhCn` | 默认 |
| 美式英语 | `enUs` | - |

## 自定义语言包

如果需要支持其他语言，可以自定义语言包：

```typescript
import { createApp } from 'vue'
import HohaUI from '@hoha/components'
import App from './App.vue'

// 自定义语言包
const customLocale = {
  name: 'ja',
  button: {
    loading: '読み込み中...',
    confirm: '確認',
    cancel: 'キャンセル'
  },
  input: {
    placeholder: '入力してください',
    clear: 'クリア'
  },
  message: {
    success: '成功',
    warning: '警告',
    error: 'エラー',
    info: '情報'
  }
}

const app = createApp(App)
app.use(HohaUI, {
  locale: customLocale
})
app.mount('#app')
```

## 动态切换语言

你可以通过 `useLocale` 钩子动态切换语言：

```vue
<template>
  <div>
    <HoButton @click="changeLocale('zh')">中文</HoButton>
    <HoButton @click="changeLocale('en')">English</HoButton>
  </div>
</template>

<script setup>
import { useLocale } from '@hoha/components'
import { zhCn, enUs } from '@hoha/components/locale'

const { setLocale } = useLocale()

const changeLocale = (lang) => {
  setLocale(lang === 'zh' ? zhCn : enUs)
}
</script>
```

## 语言包结构

```typescript
interface Locale {
  name: string
  button: {
    loading: string
    confirm: string
    cancel: string
  }
  input: {
    placeholder: string
    clear: string
  }
  message: {
    success: string
    warning: string
    error: string
    info: string
  }
  modal: {
    confirm: string
    cancel: string
    close: string
  }
  pagination: {
    total: string
    pageSize: string
    goto: string
  }
}
```

## 贡献语言包

如果你想要为 Hoha UI 贡献新的语言包，请：

1. Fork [Hoha UI 仓库](https://github.com/hoha-ui/hoha-ui)
2. 在 `packages/components/src/locale/` 目录下创建新的语言文件
3. 提交 Pull Request

我们欢迎任何语言的贡献！
