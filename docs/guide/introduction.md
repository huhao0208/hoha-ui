# 介绍

## 什么是 Hoha UI？

Hoha UI 是一个现代化的 Vue 组件库，同时支持 Vue 2 和 Vue 3。它提供了一系列高质量的 UI 组件和实用的工具函数，帮助你快速构建美观、交互丰富的 Web 应用。

## 特性

- **🔧 Vue 2/3 双版本支持** - 一套代码同时支持 Vue 2 和 Vue 3，无缝迁移
- **📦 按需引入** - 支持 Tree Shaking，只打包你使用的组件
- **🎨 丰富的组件** - 提供 Button、Input、Icon、Message、Modal、Toast 等常用组件
- **🛠️ 实用工具** - 内置 DOM 操作、格式化、存储、类型判断等工具函数
- **🌙 深色模式** - 内置深色模式支持
- **🌍 国际化** - 支持多语言配置
- **📝 TypeScript** - 完整的 TypeScript 类型定义

## 为什么选择 Hoha UI？

### 轻量级

Hoha UI 设计简洁，只包含必要的功能，打包后体积小巧。

### 易于使用

API 设计直观，文档详尽，上手简单。

### 高度可定制

支持主题定制，可以通过 CSS 变量轻松修改样式。

### 类型安全

使用 TypeScript 编写，提供完整的类型定义，获得更好的开发体验。
